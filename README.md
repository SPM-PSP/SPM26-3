# 搭子星球 - 大学生全能搭子匹配平台
本项目为四川大学《软件项目综合实践》课程设计作品，聚焦大学生日常社交、出行、学习、娱乐等场景的找搭子痛点，打造一站式全能搭子匹配平台，覆盖旅行、学习、干饭、运动、追剧等多场景，让大学生轻松找到兴趣相投、时间契合的搭子，告别独来独往。

## 项目背景
大学生群体有强烈的社交与结伴需求，但日常中普遍面临**找搭子难、兴趣不匹配、时间凑不齐、场景覆盖单一**等问题；现有社交平台要么泛社交属性过强，要么仅聚焦单一场景，缺乏针对大学生的垂直、全场景搭子匹配平台。

搭子星球精准定位大学生群体，打破单一场景限制，覆盖旅行、学习、美食、运动、娱乐等全日常场景，以**精准匹配、安全社交、场景全覆盖**为核心，打造专属于大学生的搭子匹配社交平台，让每一种需求都能找到合适的搭子。

## 核心用户画像
平台覆盖有各类结伴需求的大学生，细分为三类核心用户，精准匹配不同场景需求：
1. **社交娱乐型**
    - 核心诉求：寻找搭子一起干饭、追剧、逛街、短途旅行，丰富课余生活，拓展社交圈；
    - 行为模式：高频使用搭子发布、匹配功能，关注搭子的兴趣标签、性别、年级，追求轻松愉快的结伴体验；
    - 痛点：熟人时间难凑，找不到兴趣相投的陌生搭子，担心结伴氛围尴尬。
2. **学习成长型**
    - 核心诉求：寻找考研、考证、自习、竞赛的学习搭子，互相监督、共同进步；
    - 行为模式：精准筛选同目标、同节奏的搭子，会发布带具体时间、地点的学习结伴帖，注重搭子的执行力；
    - 痛点：独自学习缺乏动力，找不到同进度的学习伙伴，线下自习无人同行。
3. **运动体验型**
    - 核心诉求：寻找一起打球、跑步、健身、户外的运动搭子，也会寻求旅行、探店的体验型搭子；
    - 行为模式：关注搭子的技能水平、时间安排，乐于参与线下结伴活动，会分享结伴体验；
    - 痛点：独自运动枯燥，小众运动找不到同好，户外/旅行结伴有安全顾虑。

## 核心功能模块
### 首页
- 多场景搭子推荐（旅行、学习、干饭、运动等），今日热门结伴帖；
- 核心功能快捷入口（发布搭子、找搭子、看攻略/经验）；
- 全局搜索（按场景、标签、地点、时间筛选），场景分类导航。

### 找搭子
- 多维度筛选：场景、地点、时间、人数、性别、年级、兴趣标签；
- 精准匹配推荐：基于用户画像和需求智能推荐合适的搭子/结伴帖；
- 结伴帖详情：查看发布者信息、结伴要求、具体安排，支持一键申请加入。

### 发搭子
- 选择结伴场景，填写核心信息（目的地/地点、时间、人数、要求）；
- 添加兴趣标签、补充备注，支持上传图片；
- 发布后可编辑、置顶、关闭结伴帖，查看申请列表并筛选通过。

### 动态分享
- 多场景内容分类：旅行攻略、学习方法、干饭推荐、运动技巧等；
- 内容创作与分享：发布结伴体验、场景干货，支持图文/文字形式；
- 热门内容推荐，支持收藏、点赞、评论，打造优质内容社区。

### 我的
- 个人信息：学生认证、个人资料、兴趣标签、搭子等级；
- 我的结伴：我发布的、我申请的、已成功的搭子帖；
- 我的收藏：收藏的结伴帖、攻略/经验；
- 我的足迹：参与过的结伴活动，个人评价与被评价；
- 功能入口：钱包、优惠券、帮助与支持、客服中心。


## 产品核心亮点
- **全场景覆盖**：打破单一场景限制，覆盖学习、生活、娱乐、出行等大学生所有结伴需求，一个平台搞定所有搭子；
- **大学生专属**：严格学生认证，保障社交安全，贴合大学生的时间、预算、社交习惯，拒绝非目标群体干扰；
- **精准匹配**：基于兴趣标签、场景需求、时间地点、个人画像多维度匹配，告别无效社交，快速找到合适搭子；
- **社交+实用**：既满足搭子匹配的核心社交需求，又搭配场景攻略/经验分享，让结伴更省心、体验更优质。
