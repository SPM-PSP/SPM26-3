# 轻旅APP - 大学生旅行社交与规划平台
本项目为四川大学《用户界面设计》课程设计作品，聚焦大学生旅游市场痛点，打造专属的旅行社交与规划垂直平台，解决大学生旅行找伴难、预算有限、规划经验不足等问题。

## 项目背景
大学生旅游市场潜力巨大，但存在**预算有限、找伴困难、规划经验不足**等问题；传统旅游App存在内容同质化、功能复杂、消费水平不匹配等痛点，缺乏针对大学生的垂直服务平台。

轻旅APP围绕大学生核心需求，打造集**行程规划、结伴社交、攻略分享**于一体的专属平台，实现旅行更省钱、更便捷、更有趣。

## 核心用户画像
将大学生用户分为三类典型角色，精准匹配不同需求：
1. **社交主导型**：核心追求旅行结伴，拓展社交圈，痛点为熟人时间难凑、找不到兴趣相投的旅伴；
2. **预算敏感型**：极致关注性价比，痛点为担心超支、缺乏真实低消费攻略；
3. **体验探索型**：追求个性化、深度旅行体验，乐于分享，痛点为大众攻略无法满足需求、独自探索有安全顾虑。

## 核心功能模块
### 首页
- 推荐高性价比行程、今日旅游优惠（学生证折扣/特价门票）
- 核心功能快捷入口（生成行程/找同伴/看攻略）
- 全局搜索入口、热门结伴推荐

### 行程
- 行程创建/筛选、自定义规划功能
- 行程详情时间轴视图
- 大学生专属行程模板推荐（穷游/短途/城市深度等）

### 结伴
- 发布结伴帖（选择目的地、时间、人数、标签）
- 结伴信息筛选/搜索、申请加入功能
- 热门结伴推荐，基于兴趣标签匹配旅伴

### 攻略
- 攻略分类浏览、城市攻略聚合
- 攻略创建/发布、图文分享功能
- 热门攻略推荐，支持搜索

### 我的
- 个人信息（学生认证、等级）
- 个人资产（行程/结伴/攻略）
- 收藏/足迹/订单/优惠券管理
- 帮助与支持、客服入口


## 产品核心亮点
- **大学生专属**：贴合学生预算与出行习惯，提供学生证优惠、拼房拼车等专属信息；
- **社交核心**：打造安全可靠的结伴系统，基于兴趣标签匹配旅伴，解决“和谁玩”的核心痛点；
- **高性价比**：提供定制化穷游模板、真实学生省钱攻略，实现预算内最优旅行体验；
- **个性化规划**：支持自定义行程设计，满足不同用户的旅行探索需求。
